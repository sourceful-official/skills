# Sourceful Agent Skills

This repo contains Agent Skills for Sourceful, focused on Riverflow 2.0 launch partners.

## Skills

- `skills/riverflow-2-partners-api` — Integration guidance for Riverflow 2.0 launch partners (Replicate, Runware, OpenRouter).
- `skills/riverflow-2-best-practices` — Prompting and creative workflow guidance for Riverflow 2.0.

## Structure

Each skill folder contains a `SKILL.md` and optional `references/` or `assets/` directories per the Agent Skills specification.
